Chapter 966: Mengumpulkan Petir Mutlak

Jiwa naga iblis itu terhubung dengannya, jadi ia juga mengambil bentuk manusia. Ia berkata dengan terkejut, “Bahkan kamu tidak dapat menemukannya. Mungkinkah kedua api itu saling mengikis dan menghilang?”

kelima elemen tersebut saling memperkuat dan melawan,” kata Yunxiao serius. jika ada dua atribut dari lima elemen yang saling berlawanan, maka situasi yang kamu sebutkan sangat mungkin terjadi. Namun, kekuatan kedua api tersebut tidak akan saling meniadakan. Apalagi api es tersebut hanyalah nyala api kecil yang berasal dari batu pelangi, bukan kekuatan api aslinya. Di sisi lain, api ilahi Phoenix adalah gumpalan api asal, sehingga harus mampu melahap lawan secara langsung. Ini juga alasan mengapa aku mengambil risiko menggunakan api asal Phoenix.”

Naga Iblis itu mengerutkan kening dan berkata, “Tetapi faktanya kedua api itu tidak ada sekarang. Ataukah api es itu bahkan lebih tinggi peringkatnya daripada api ilahi Phoenix?” Nyala api kecil akan mampu memadamkannya dan juga meninggalkan nyala api kecil pada binatang batu itu.”

Yunxiao menggelengkan kepalanya. itu tidak mungkin. Bahkan di antara roh sejati tingkat sepuluh, burung phoenix berada di peringkat beberapa teratas. Jika aku tidak salah, gumpalan api asli itu mungkin sedang berevolusi.

“Evolusi?”

Apa? Naga iblis itu terkejut. itu sudah menjadi item tingkat sepuluh. Masih bisa berkembang?”

meskipun aku belum mencapai tingkat kesepuluh,” kata Yunxiao, “Karena roh sejati kuno memiliki tingkat kekuatan yang berbeda, mereka secara alami telah berevolusi juga. Alasan mengapa aku memiliki dugaan seperti itu adalah karena kekuatan dunia tablet alam dewa tidak melemah, tetapi tampaknya sedikit meningkat. Kekuatan api ilahi mungkin cukup untuk menembus kekuatan seluruh dunia, jadi meskipun aku adalah pemilik tempat ini, aku tidak dapat memata-matai keberadaannya selama dia tidak mau.”

Yunxiao menatap Labu Little King Kong itu lagi dan berkata, “Benih api di tubuh Labu Little King Kong seharusnya adalah sisa setelah Api Phoenix melahapnya hingga kapasitas penuhnya. Bagaimanapun, hanya ada satu gumpalan api aslinya, yang terlalu kecil.

“Jika itu masalahnya, bukankah itu berarti selama aku bisa mengendalikan api es ini, aku bisa terus memasukkannya ke Api Phoenix Sejati dan membuatnya lebih kuat?” Naga Iblis bertanya dengan ekspresi aneh.

seharusnya begitu,” kata Yunxiao. tapi, api es pelangi ini terlalu jahat dan menakutkan. Jika itu adalah kekuatan penuh dari api aslinya, mungkin saja itu berada di atas Api Phoenix Sejati. Bagaimana cara mengendalikannya?”

Saat berikutnya, tubuhnya berkedip dan dia muncul di samping Xiao Hong.

Xiao Hong masih melayang di udara dengan tenang. Cahaya di sekitar tubuhnya tidak melemah sama sekali, seperti seorang putri yang sedang tidur.

Namun, auranya membuat Yunxiao merasakan ketakutan yang tidak bisa dijelaskan, dan dia tidak berani mendekat.

Aegirine, yang sebelumnya ingin dihilangkan oleh Penguasa Kota Bulan Merah tanpa alasan, dan batu pelangi, yang dianggap sangat penting oleh istana Dong Hai, akan membawa perubahan besar ketika keduanya digabungkan. Jika bukan karena tablet alam dewa, Yunxiao tidak akan berani mengambilnya.

Ekspresi Yunxiao berubah beberapa kali sebelum dia keluar dari tablet alam dewa. Tubuh utamanya, yang telah mengatur pernapasannya, sedikit membuka matanya.

Botol Jade yang ramping tergantung di udara, memancarkan lapisan kilau. Perjuangan api Phoenix di bawahnya semakin lemah, dan secara bertahap diserap.

“Mengaum!”

Naga Petir di kejauhan meraung dengan marah. Lingkaran petir mengalir deras dan menerobos kurungan ketiga anggota keluarga Wei dalam sekejap.

Kedua makhluk unsur itu telah bersama selama ribuan tahun. Meski kecerdasan mereka rendah, ada emosi yang tidak bisa dijelaskan di antara mereka. Phoenix api akan menderita, dan Naga Guntur mengamuk tanpa peduli.

Ekspresi Wei Dongyang tiba-tiba berubah dan dia berkata dengan marah, “Aku bahkan tidak bisa menjebak sebuah elemen!”

Dia mengarahkan jarinya ke udara, dan pedang di tangannya memancarkan cahaya cemerlang. Cahaya pedang menyebar seperti sayap, dan pedang Qi meluncur keluar seperti binatang yang dingin dan ganas.

“Mengaum!”

Naga Petir terpotong oleh cahaya pedang, dan momentumnya segera melemah. Warna hijau menjadi redup.

Phoenix api, yang hendak disedot ke dalam botol Giok, merasakan sesuatu dan membalikkan tubuhnya ke udara. Tubuhnya yang putih menjadi lebih cerah dan titik di antara alisnya menjadi semakin cerah.

“Tuan pemimpin klan, ada situasi!” Seorang anggota keluarga Wei tiba-tiba berseru.

Wei Dongyang hendak menebas Naga Petir itu lagi ketika dia tiba-tiba tersentak dan berbalik.

Phoenix api menjerit sedih, dan nyala api di antara alisnya tiba-tiba meledak. Seluruh tubuhnya langsung berubah menjadi ketiadaan, dan api putih keluar.

“Bang!”

Botol Giok di langit adalah yang pertama menanggung beban terberat dan meledak.

Hati Wei Dongyang sakit, tetapi sebelum dia bisa marah, nyala api putih menyerbu. Dia memegang pedangnya secara horizontal di depannya karena terkejut dan marah, dan penghalang pedang segera muncul.

“Bang! Bang! Bang! Bang!”

Suara ledakan datang dari segala arah. Meskipun penghalang Wei Dongyang tidak pecah, dia langsung didorong mundur puluhan meter oleh kekuatan tersebut. Nyala api yang memenuhi langit hampir membakar seluruh dasar danau hingga menjadi ruang hampa.

“Hualala!”

Air danau dalam jumlah besar memenuhi area tersebut, dan seluruh pemandangan menjadi kacau balau.

Semua anggota keluarga Wei terluka oleh api yang menghancurkan diri Phoenix. Salah satu dari mereka berada paling dekat dengan api dan berteriak kesakitan. Pada akhirnya tidak ada suara dan dia terbakar menjadi abu di dasar danau.

Semua orang tampak ketakutan. Wei Dongyang sangat marah hingga tubuhnya gemetar. Dia tidak hanya kehilangan botol Giok dan nyawa Martial Supreme, tapi dia juga memberi Marsekal Wei harta yang berharga. Pada akhirnya, dia telah melukai semua orang dan tidak mendapatkan apapun!

Di bawah pengaruh api, Naga Petir menjadi redup dan mengeluarkan tangisan sedih.

Mata Yunxiao bersinar saat dia melakukan gerakan mantra dengan satu tangan.

Naga iblis segera bergabung dengan jiwanya, dan tubuhnya mulai menjadi iblis. Kekuatan jiwa yang kuat menyebar, dan dia berteleportasi dalam sekejap.

Marsekal Junwei sangat terkejut. Dia sangat senang melihat keluarga Wei menderita kerugian besar sehingga dia tidak memperhatikan kondisi Yunxiao. Sebaliknya, dia merasakan kekuatan spiritual yang kuat dan menemukan bahwa Yunxiao telah berubah menjadi iblis.

Setelah memurnikan Labu Little King Kong, kekuatan jiwa Yunxiao tidak dapat dipulihkan dalam waktu singkat, jadi dia hanya bisa mengandalkan kekuatan Naga iblis, yang merupakan Jiwa Naga tingkat sembilan, dan kekuatannya tidak lebih tinggi dari milik Yunxiao.

Dia membuat mantra petir dengan kedua tangannya dan turun dari langit, menjebak Naga petir di dalamnya.

Naga Petir sudah menjadi anak panah di akhir penerbangannya, jadi bagaimana ia bisa menahannya? di bawah pengawasan orang banyak, itu langsung tersedot ke dalam tubuh Yunxiao.

Pemandangan ini menarik perhatian semua orang. Semua anggota keluarga Wei saling memandang dengan bingung, tidak tahu apa yang mereka rasakan.

Wei Dongyang, sebaliknya, sangat marah hingga dia akan meledak. Mereka berdua mengumpulkan roh unsur, tapi hasilnya sangat berbeda. Matanya memuntahkan api saat dia menatap Yunxiao, dorongan untuk bergegas dan membunuhnya muncul di dalam hatinya.

haha, selamat Tuan Yunxiao karena berhasil memperoleh petir mutlak sembilan Yin. Selamat, selamat!

Wajah Marsekal Junwei penuh dengan senyuman saat dia bersorak keras. Kemudian, dia menghela nafas dan berkata, sayang sekali koleksi saudara Dong Yang tidak begitu mulus. Sayang sekali!

Bibir Wei Dongyang bergerak sedikit seolah ingin mengatakan sesuatu, tapi dia menahannya. Ekspresinya terlihat seperti hendak menangis, membuat Marsekal Junwei merasa senang.

Seorang pejuang keluarga Wei akhirnya tidak bisa menahan diri untuk tidak berkata dengan marah, “Yunxiao, kami melukai serius Sembilan Guntur mutlak Yin, itulah sebabnya kamu menganggapnya begitu mudah. Kamu tidak dapat mengambilnya sendiri!

Anggota keluarga Wei lainnya juga mengungkapkan kesedihan di wajah mereka. Salah satu dari mereka berkata, “Kami juga kehilangan seorang ahli. Kamu tidak menyumbangkan apapun pada petir absolut sembilan Yin, jadi itu semua seharusnya menjadi milik keluarga Weiku.

Wei Dongyang terus menganggukkan kepalanya. Meskipun dia tidak mengungkapkan pendapatnya, dia jelas memiliki niat yang sama.

“Apakah kamu masih merasa malu?” Shuai Junwei berteriak dengan marah. Pada awalnya, aku mengatakan bahwa sembilan petir mutlak Yin adalah milik Tuan Yunxiao, dan delapan Api Gelap Yin adalah milikmu!”

Dia tampak sangat marah di permukaan, tetapi sebenarnya dia tertawa di dalam hatinya, berharap hubungan mereka dengan Yunxiao akan hancur total.

Seorang pejuang mendengus dingin dan berkata, “Itulah yang kita sepakati sebelumnya, namun kenyataannya, kita harus memberikan kontribusi yang nyata. Kami, keluarga Wei, hampir melakukan yang terbaik, sedangkan Li Yunxiao tidak melakukan apapun. Bagaimana dia bisa mendapatkan Guntur mutlak?”

Wajah Yunxiao menjadi dingin saat dia berkata, “Aku tidak mengatakan apa-apa karena aku ingin semua orang bekerja sama dan membantu satu sama lain, tetapi karena kamu sudah mengatakannya, aku harus menyelesaikan masalah ini denganmu!”

Dia berkata dengan dingin, “Jika sembilan petir absolut Yin ini diambil olehku dalam kondisi puncakku, efeknya akan sepuluh kali lipat dari kondisi melemahnya saat ini!” Kami telah sepakat bahwa kalian akan mengambil Api gelap dan aku akan mengambil petir absolut, tetapi kalian menyerang Naga Petir absolut sembilan Yinku dan membuatnya sangat lemah. Tidak ada gunanya mengambilnya kembali sekarang, dan kamu masih memiliki wajah untuk berbicara denganku tentang petir mutlak? Tuan Dong Yang, bisakah kamu memberiku penjelasan atas pelanggaran kontrak ini?”

Semua orang berkeringat dingin. Kalau begitu, keluarga Wei-lah yang berhutang pada Yunxiao.

Meskipun kedengarannya agak keterlaluan karena Yunxiao tidak akan bisa mendapatkan Sembilan Yin Guntur Mutlak dengan mudah tanpa orang-orang dari keluarga Wei, apa yang dikatakannya masuk akal. 'Karena kita sudah mencapai kesepakatan sejak awal, apa benar? apakah kamu harus menyentuh barang-barangnya?'

Shuai Junwei berpura-pura terkejut dan berteriak dengan ekspresi berlebihan, “Jadi begitu! Ya, saudara Dongyang, tolong beri penjelasan pada Tuan Yunxiao!”

Kemarahan Wei Dongyang berubah menjadi keringat. Kata-kata Yunxiao memang sempurna. Mereka sendiri yang memprovokasi Naga Petir, jadi siapa yang bisa mereka salahkan? Dia sangat tidak rela dan berkata, “Jika tidak ada orang dari keluarga Wei, bagaimana mungkin Tuan Yunxiao bisa dengan mudah mendapatkan petir absolut? ini bisa dianggap sedikit membantu, bukan?”

Hmph! Yunxiao berkata, “Aku tidak bisa mendapatkan Guntur mutlak tanpamu? Bukankah kalian semua melebih-lebihkan diri kalian sendiri? Semua spekulasi hanyalah angan-anganmu sendiri, dan fakta bahwa kamu menghancurkan Naga Petirku adalah fakta yang sangat kuat! Hehe, Tuan Dongyang, kamu menggunakan tebakanmu untuk memverifikasi kebenaran. Kamu menindasku karena tidak cukup membaca!”

Wei Dongyang merasa pusing dan berkata dengan marah, “Sekarang kita sudah bertarung, apa yang kamu inginkan?”

Marsekal Junwei mencibir, “Hehe, kamu tidak masuk akal, jadi kamu menggunakan kekerasan? Keluarga Shuai-ku pasti akan berdiri di sisi Tuan Yunxiao!”

Tubuh Wei Dongyang gemetar karena marah, dan niat membunuh terus muncul di matanya. Jika mereka bertarung dengan keluarga Shuai sejak awal, mereka akan memiliki peluang menang yang besar. Sekarang mereka telah kehilangan tiga orang dan semuanya terluka, mereka benar-benar dirugikan dan tidak dapat melawan keluarga Shuai.

Wei Dongyang tiba-tiba merasa keadaan menjadi semakin buruk, dan dia kehilangan inisiatif sedikit demi sedikit.
