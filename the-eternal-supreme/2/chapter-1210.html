Chapter 1210: Setelah Berakhirnya Ilmu Bela Diri Dao

Li Yunxiao terkejut. "Mungkinkah Istana Kaisar Laut memiliki karakter sebenarnya dari 2.640 karakter sisa yang terukir di pilar luar?"

Li Yunxiao sangat terkejut di hatinya, dan suaranya bergetar ketika dia berkata, "Jika ... jika kekuatan karakter kuno ini sepenuhnya dipahami, keberadaan seperti apa yang akan terjadi?" Dia akhirnya menanyakan pertanyaan yang paling ingin dia ketahui, "Apakah masih mungkin untuk melangkah ke Alam Ilahi Sepuluh Arah di bawah langit ini?"

Ekspresi Kaisar Laut Bronn menjadi agak serius, dan dia mengubah topik, "Tahukah kamu apa yang terjadi setelah puncak Sembilan Bintang?"

Li Yunxiao berkata tanpa ragu-ragu, "Sebelum akhir seni bela diri, melampaui kematian dan masuk ke dalam kesucian."

Kaisar Laut mengangguk, "Kamu setengah benar. Setelah puncak Sembilan Bintang, memang benar bahwa seseorang akan melampaui kematian dan memasuki kesucian. Namun, itu bukan sebelum akhir dari seni bela diri, tetapi setelahnya."

"Apa? Setelah seni bela diri berakhir?"

Kaisar Laut berkata dengan suara yang dalam, "Dahulu kala, memang seperti ini. Setelah akhir seni bela diri adalah Alam Ilahi dari Sepuluh Arah, yang secara langsung memasuki tingkat ketuhanan. Dia mengangkat matanya dan menatap Li Yunxiao, "Ketika kamu berada pada kondisi terkuat di kehidupan sebelumnya, kamu seharusnya juga melangkah ke alam transendensi dan kesucian, kan? Kaisar Laut berkata, "Tidak mengherankan jika Anda memiliki pemikiran seperti ini. Di bawah langit ini, masih ada peluang untuk menjadi dewa, tetapi ini sangat keras. Misalnya, jika Anda dapat memulihkan dan menguasai Maha Kuno Kitab Suci, tidak peduli bagaimana dunia berubah, itu tidak akan bisa menghentikanmu untuk menjadi dewa."

Li Yunxiao terdiam. Dia tahu bahwa Kaisar Laut tidak akan dengan mudah mengungkapkan rahasia seperti itu. Bagaimanapun, ini terkait dengan impian utama para ahli yang tak terhitung jumlahnya selama puluhan ribu tahun. Adapun pemulihan Kitab Suci Maha Kuno, itu mustahil.

Ia berkata, "Jika saya melengkapi kata-kata pada pilar di luar aula, dapatkah Raja Kaisar Laut memberi tahu saya apa yang Anda ketahui tentang jalan menuju keilahian?"

Kaisar Laut tersenyum, "Kamu tidak dapat menyelesaikannya. Tiga ratus enam puluh kata yang hilang didistribusikan ke empat seni dewa. Kamu hanya dapat menyelesaikan seratus delapan puluh kata. Sebagai hadiah, saya dapat membiarkan kamu memilih lima ratus kata dari pilar untuk dipahami."

Hati Li Yunxiao tergerak. Lima ratus kata bukanlah jumlah yang kecil. Jika dia bisa memahaminya sepenuhnya, manfaatnya akan beragam.

Kaisar Laut terkekeh, "Kamu tidak perlu terburu-buru menjawabku. Kamu bisa meluangkan waktu untuk memikirkannya."

Matanya berangsur-angsur bersinar dan menjadi sedikit berapi-api. â€œTuan Muda Yunxiao, bisakah Anda menunjukkan kepada saya api dewa peringkat sepuluh sekarang?â€

Ini adalah pertama kalinya wajahnya yang awalnya tenang mengalami perubahan besar. Itu membuktikan bahwa dia tidak bisa menyembunyikan kegembiraan dan kegembiraan yang luar biasa di dalam hatinya.

Li Yunxiao mengangguk dan tidak menahan diri lagi. Dia membentuk segel dengan satu tangan dan tanda phoenix muncul di antara alisnya. Kemudian, kicauan kuno terdengar dan nyala api seukuran ibu jari berkedip di antara alisnya.

"Api Phoenix Sejati!"

Kaisar Laut berteriak kaget. Wajahnya penuh kegembiraan, tapi kemudian dia dengan hati-hati menatap nyala api dan perlahan mengerutkan kening.

Li Yunxiao berkata, "Apa, nyala api ini tidak cukup?"

Kaisar Laut berkata, "Ini memang Api Phoenix Sejati yang sebenarnya, tetapi nyala api ini terluka."

"Terluka?"

Jantung Li Yunxiao berdetak kencang. Ketika dia mendapatkan True Phoenix Flame, dia langsung menerima informasi yang terkandung di dalamnya. Divine Flame Phoenix memang terluka parah dalam pertempuran dengan Black Flame Evil Soul. Dia tidak bisa menahan rasa terkejutnya. Raja Laut bahkan bisa melihat ini?

Kaisar Laut mengangguk dan berkata, "Ya, dan nyala api ini agak aneh. Sepertinya telah dimodifikasi oleh nyala api luar?" Wajahnya penuh keraguan. â€œAneh, api macam apa di dunia ini yang bisa menyebabkan Api Phoenix Sejati berubah?â€

Li Yunxiao tidak bisa tidak yakin. Ia berkata, "Mata Kebenaran memang merupakan mata dewa terkuat di dunia. Api Phoenix Sejati ini telah menyerap sedikit Api Hati Iblis Es."

Ekspresi Kaisar Laut berubah drastis dan memperlihatkan ekspresi terkejut. Dia tiba-tiba berkata, â€œJadi begitu. Pantas saja, tidak heranâ€¦â€

Dia bergumam pada dirinya sendiri beberapa saat sebelum berkata, "Sayang sekali. Jika saya bisa menyerap lebih banyak lagi, mungkin api ilahi ini bisa memperbaiki dirinya sendiri."

Mata Li Yunxiao bergerak. Api ilahi di antara kedua alisnya berangsur-angsur padam. Dia berkata, "Mungkinkah nyala api ini tidak dapat menyalakan cahaya Bintang Tujuh?"

Kaisar Laut berkata, "Kecuali jika Anda bisa tinggal di Istana Kekaisaran Laut untuk waktu yang lama dan menjaga api ilahi."

Wajah Li Yunxiao menjadi gelap. Ini jelas merupakan kondisi yang mustahil.

Kaisar Laut tersenyum, "Baik, sebagai hadiah karena telah membantuku, aku akan memperbaiki api ilahi tingkat sepuluhmu. Bagaimana menurutmu?"

Untuk mengembangkan seni dewa ini, nenek moyang saya pernah memperoleh benda aneh yang disebut Tungku Duniawi. Terdapat sejumlah besar energi vital elemen api di dalam yang dapat menyuburkan api ilahi. Murid tertua saya, Gu Hong, baru saja keluar dari pengasingan."

Li Yunxiao bertanya, "Berapa lama waktu yang dibutuhkan untuk memperbaiki api ilahi?"

Kaisar Laut berkata, "Itu tergantung pada kendali Anda atas api ilahi. Belum pernah ada preseden seperti itu. Saya juga tidak tahu. Ikuti saya."

Sosok Kaisar Laut berangsur-angsur memudar seperti bulan di dalam air. Dia berubah menjadi seberkas cahaya dan melesat keluar istana.

Li Yunxiao dan Ye Fan akhirnya menyadari bahwa Kaisar Laut di depan mereka hanyalah proyeksi. Tubuh aslinya mungkin masih menjaga cahaya Bintang Tujuh.

Mereka berdua segera menyusul, mengikuti pancaran cahaya yang melintasi seluruh pulau.

Pulau itu dipenuhi kicauan burung dan harumnya bunga. Tampaknya akrab dengan aura para kultivator ini. Tiga sinar cahaya lewat tanpa panik. Suasananya tenang dan damai.

Tak lama kemudian, mereka sampai di sudut pulau, menghadap laut dalam yang tak berujung.

Proyeksi Kaisar Laut muncul. Dia berjalan di depan laut dalam dan menatap ke bawah.

Li Yunxiao dan Ye Fan juga menunduk. Mereka melihat laut di depan mereka berwarna merah tua. Gelombang panas yang bergulung-gulung menerpa wajah mereka.

Ye Fan berkata dengan kaget, "Ini benar-benar lautan api."

Ombak merah terus menerjang pantai. Setiap kali mereka menyerang, suhunya sangat tinggi. Seolah-olah mereka terkena sinar matahari.

Kaisar Laut berkata, "Tungku Duniawi ada di bawah sini. Tungku Duniawi itu ditindas oleh leluhur dengan es yang berumur sepuluh ribu tahun."

Li Yunxiao bertanya dengan ragu, â€œBisakah tungku ini hanya menampung satu orang?â€

Kaisar Laut berkata, "Tentu saja tidak. Maksudmu murid tertuaku, Gu Hong?"

Ekspresi aneh muncul di wajahnya. Ia berkata, "Sebaiknya kau tidak menemuinya. Tunggu hingga api ilahi pulih dan cahaya Bintang Tujuh menyala. Lalu kau boleh pergi."

Li Yunxiao tidak mengerti. "Mengapa?"

Kaisar Laut berkata, "Karena Gu Hong sangat mencintai Narcissus. Sudah lama sekali, dan Narcissus sekarang mencintaimu. Jika Gu Hong tahu, dia pasti akan membunuhmu."
