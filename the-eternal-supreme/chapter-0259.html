Chapter 259: Wilayah Kebakaran

“Jadi, ini adalah dunia api?” Duan Yue mengerutkan kening. Dia tidak bisa merasakan sesuatu yang berbeda. Terlepas dari gumpalan energi roh yang melayang di udara, yang ada hanyalah keheningan.

Yunxiao menutup matanya sedikit, mengangkat satu tangan, dan menempelkannya ke dadanya sambil berbisik, “Tiba-tiba, karakter seperti kecebong emas muncul. Itu adalah teknik dewa alam yang agung. Seluruh tubuhnya mulai bersinar dengan cahaya keemasan, dan fenomenanya persis sama seperti saat dia menggunakan teknik dewa ekspansi besar.

Satu demi satu, kata-kata tidak jelas keluar dari mulut Yunxiao, membuatnya tampak serius dan sakral, seperti dewa yang turun ke dunia.

Duan Yue terkejut. Di tablet alam dewa ini, Yunxiao adalah Dewa yang tak tertandingi. Saat Kitab Suci yang tidak jelas itu dibacakan, ruangan itu mulai bergetar seolah-olah sebuah pintu kuno perlahan-lahan terbuka, dan terdengar suara gemuruh yang tumpul.

Sebuah pola perlahan muncul di kerikil gurun yang tak terbatas. Tidak diketahui berapa panjang dan lebarnya, tapi itu adalah pemandangan yang menakjubkan, seolah-olah seekor binatang purba tertidur di bawah tanah!

Ketika Yunxiao membuka matanya, dia juga terkejut dengan susunan megah di depannya. Meskipun dia berdiri tinggi di langit dan melihat ke bawah ke bumi, dia masih merasa sangat kecil dan tidak terjangkau.

“Seperti yang diharapkan dari formasi yang bisa membuka suatu wilayah. Sungguh luar biasa!”

Yunxiao, yang cukup percaya diri, mau tidak mau merasakan ketidakberdayaan ketika dia melihat tampilan sebenarnya dari array tersebut. Dia menyentuh ratusan juta batu primordial kelas menengah yang dia miliki, tapi sepertinya itu tidak cukup untuk mengisi celah di antara gigi monster formasi ini.

Duan Yue juga merasa sedikit sedih, jadi dia bertanya dengan hati-hati, “Apakah kamu punya cukup batu purba?”

itu seharusnya cukup untuk mengisi celah di antara gigiku,” kata Yunxiao sambil menyeka keringat dinginnya.

Duan Yue, “……”

tidak apa-apa,” kata Yunxiao sambil tersenyum masam. Aku akan mengaktifkan arraynya terlebih dahulu. Dengan nyala api Phoenix Sejati di mata array, seharusnya tidak menjadi masalah besar untuk mengaktifkannya saja. Akan merepotkan jika ingin mengoperasikannya.”

Dia menyentuh ruang di antara alisnya dan api yang sebenarnya muncul. Itu membentuk api Phoenix di udara dan terbang menuju pusat formasi.

Duan Yue merasakan suhu yang sangat panas, dan Api Bumi teratai biru yang dimurnikan dalam dantiannya benar-benar mengeluarkan ratapan ketakutan.

Di bawah api ilahi, semua api akan tunduk!

Sayangnya, dengan basis kultivasi Yunxiao saat ini, dia hanya dapat mewujudkan api ilahi di tablet alam ilahi. Jika tidak, tak seorang pun di seluruh wilayah selatan benua surgawi Martial akan mampu menandingi api ilahi.

Api Phoenix melesat melintasi langit seperti meteor dan jatuh ke tanah. Itu mendarat di inti susunan dengan ledakan keras dan menyebabkan api membumbung ke langit.

Seluruh langit bersinar merah, seolah-olah itu adalah akhir dunia, dan mereka berada di lautan api neraka.

Yunxiao telah mengalami perasaan ini sekali, di dunia kesadaran roh sejati Phoenix di Gunung Phoenix di Tianshui. Api Nirwana semacam inilah yang memungkinkannya terlahir kembali setelah bencana.

Wajah Duan Yue telah lama menjadi pucat di bawah ancaman langit dan bumi, dan dia mau tidak mau ingin berlutut dan beribadah.

“Tenang!”

ayo pergi! Yunxiao berteriak sambil terbang ke bawah seperti ROC dengan sayap terbentang, jatuh ke dalam api.

Tangisan lembut terdengar di benak Duan Yue seperti genderang malam dan bel pagi, menariknya kembali dari rasa takut. Duan Yue berkeringat dingin. Dia berdiri di udara dan mengedarkan energi aslinya untuk melawan kekuatan langit dan bumi.

Saat Yunxiao jatuh ke lautan api, nyala api di dunia menjadi sedikit lebih lembut, tidak lagi panas dan kuat.

Tubuh Yunxiao dikelilingi oleh cahaya keemasan, dan matanya juga keemasan, dengan karakter seperti kecebong yang terus-menerus muncul dan menghilang. Saat dia memukul simbol-simbol itu satu per satu, api persegi itu perlahan-lahan menjadi jinak. Tampaknya dipandu oleh kekuatannya dan menyebar ke dalam barisan.

Ketika Duan Yue merasakan tekanan langit dan bumi berangsur-angsur melemah, dia membuka matanya dengan rasa takut yang masih ada, hanya untuk melihat lautan api Merah di bawah, dan Yunxiao telah lama menghilang. Dia mengetahui kemampuan Yunxiao, jadi dia tidak terlalu khawatir. Pada saat ini, Api Bumi Teratai Biru di tubuhnya sudah mulai beradaptasi dengan perasaannya, perlahan menyala kembali, dan semakin membesar.

Duan Yue sangat gembira. Di bawah pengaruh api ilahi Phoenix, Api Bumi teratai biru tampaknya menunjukkan tanda-tanda evolusi. Dia buru-buru menyembunyikan roh dan Qi-nya dan mulai perlahan-lahan menyerap elemen api di dunia ke dalam tubuhnya, dengan hati-hati mengolah Api Bumi teratai biru. Api ini adalah api langit dan bumi yang menakjubkan, dan telah dimurnikan menjadi satu dengannya. Keduanya maju dan mundur bersama.

Waktu tidak terbatas di tablet alam ilahi, dan tidak ada perubahan siang dan malam. Sepertinya beberapa hari telah berlalu.

Duan Yue perlahan membuka matanya, yang merah karena api dan bersinar karena terkejut. Api Bumi Bluelotus miliknya sebenarnya telah berevolusi dalam beberapa hari terakhir, dan kekuatannya jauh lebih besar dari sebelumnya.

Dia melirik lautan api di bawah, yang masih terus membesar, dan sedikit kekhawatiran muncul di hatinya, takut Yunxiao akan terkejut.

Tiba-tiba, api di bawah mulai melonjak dan berkontraksi dari segala arah, menyatu menuju pusat formasi. Dalam sekejap mata, api sepanjang sepuluh ribu mil menyatu menjadi satu garis dan dibawa ke telapak tangan manusia. Itu adalah tarian iblis.

“Tuan Muda Yun!”

Yunxiao! Duan Yue berteriak kaget. Tubuh Yunxiao tampak sedikit membesar, dan otot-ototnya menonjol, membuatnya tampak seperti pria besar. Itu adalah Supreme tubuh tiran dari teknik penempaan tubuh tiran.

Dengan ekspresi serius di wajahnya, Yunxiao perlahan menyeret api di tangannya dan memasukkannya ke dalam lubang susunan. Dalam sekejap, seluruh susunan berubah menjadi merah menyala, seperti binatang purba yang tertidur di tanah.

Duan Yue buru-buru terbang ke bawah dan mendarat di samping Yunxiao, lalu bertanya dengan gugup, “” Bagaimana kabarnya?”

Aku telah mengaktifkan susunannya, tetapi sangat tidak stabil,” kata Yunxiao serius. tanpa suntikan semangat Qi, cepat atau lambat akan mengering lagi. Aku berencana menambahkan teknik formasi untuk menstabilkan formasi kuno ini.”

Kehendak spiritualnya bergerak, dan udara segera berubah. Dalam sekejap, sejumlah besar orang muncul di udara. Mereka adalah ribuan Prajurit yang berkultivasi di tablet alam dewa.

Duan Yue dikejutkan oleh kemunculan ribuan orang secara tiba-tiba.

Semua orang juga bingung. Mereka telah berkultivasi dengan baik, jadi mengapa mereka tiba-tiba muncul di tempat yang tidak dapat dijelaskan ini?

Semuanya, tenang!

Suara Yunxiao terdengar, dan semua orang segera mengerti apa yang sedang terjadi, dan mereka berhenti berbicara.

Selama periode waktu ini, setiap orang telah memperoleh manfaat besar, dan kecepatan kultivasi mereka sangat cepat. Mereka semua mengagumi Yunxiao dan sangat yakin.

Berdiri di udara, Yunxiao mengeluarkan aura Grandmaster sambil berkata perlahan,”“Roh Qi di tablet alam dewa semakin tipis, dan aku khawatir itu tidak akan cukup untuk kultivasimu. Aku akan membangun lautan energi spiritual sementara di area ini dan mengarahkan semua energi spiritual ke tempat ini. Pada saat yang sama, aku membutuhkan 3.600 orang untuk mempertahankan formasi untukku, yang akan sangat bermanfaat bagi kultivasimu. Semua orang akan bermain secara bergiliran.”

Kelompok artis Martial semuanya sangat terkejut dan bingung.

Yunxiao mengeluarkan beberapa tas penyimpanan dari cincinnya, yang berisi ratusan juta batu primordial kelas menengah, lalu melemparkannya ke udara dan melayangkan pukulan.

“LEDAKAN!”

Kantong penyimpanan meledak satu demi satu, dan batu primordial di dalamnya semuanya hancur oleh angin dari kepalan tangan. Sejumlah besar Qi vital yang mengejutkan menyebar, begitu padat hingga hampir tidak terlihat.

“?!”

Semua orang terkejut. Qi vital yang begitu padat hampir menghanyutkan mereka. Hanya dengan bernapas beberapa kali, dia merasa segar dan kultivasinya meningkat.

“Haha, ini sangat memuaskan! Aku benar-benar berhasil menerobos dalam sekejap!”

sialnya, Qi vital ini seperti air. Aku merasa kenyang.

Semua orang sangat gembira dan buru-buru mulai menyerapnya. Banyak artis Martial yang terjebak dalam kemacetan secara kolektif berhasil menerobos saat ini.

Yunxiao mengeluarkan bendera kecil lainnya, matanya bersinar. Itu adalah bendera perang tianluo yang dia menangkan dari The Alchemist Association. Meskipun rusak parah, itu adalah artefak asli tingkat sembilan!

Di tangan GE Xiangqian, benda ini hanya memiliki nilai penelitian, tetapi Yunxiao dapat melihat bahwa bendera pertempuran belum mati. Dengan kemampuannya, dia secara alami punya cara untuk merangsangnya. Karena itu juga merupakan formasi atribut api, itu adalah yang paling cocok untuk ditambahkan ke formasi domain api.

Karakter emas berkedip-kedip di matanya dan melesat ke udara, terpantul pada spanduk perang api tianluo yang menyala-nyala, dan ditembakkan.

Pola compang-camping pada bendera pertempuran ditutupi dengan lapisan cahaya keemasan, dan kemudian perlahan-lahan memuntahkan cahaya merah menyala. Seluruh bendera pertempuran terbakar di udara, seolah-olah akan berubah menjadi abu. Namun, hal yang mengejutkan adalah apinya tidak hanya hilang, tetapi malah semakin membesar. Dalam sekejap mata, itu berubah menjadi lautan api, bersiul dan menyelimuti semua seniman Martial.

Pada saat yang sama, Yunxiao dengan cepat mengukir susunan di udara. Dengan kilatan tangannya, diagram susunan kecil mengembun di udara dan membentuk pola yang lebih besar, yang kemudian ia lemparkan ke lautan roh Qi.

Formasi itu terbang ke dalamnya dan segera berubah menjadi cahaya keemasan yang menyebar. Segera, ia menyelimuti seluruh lautan energi spiritual dan menekan kebocoran energi spiritual.

Panji perang api yang berkobar di tianluo meluas dengan cepat, dan lautan api turun dari langit, menutupi formasi kuno. Di mata barisan, avatar api primordial yang dibuat Yunxiao dengan cepat tumbuh lebih kuat di bawah lautan api bendera pertempuran.

Sekarang aku akan mengajarimu teknik menggunakan spanduk. Kultivasimu terlalu lemah, jadi kamu memerlukan kekuatan gabungan 3600 orang untuk menggunakannya.

Saat Yunxiao berbicara, satu demi satu simbol yang tidak jelas mengembun di udara dan tidak menghilang. Tidak ada gelombang suara yang dikirim, tetapi gelombang itu membekas di hati setiap pejuang dan mengalir langsung ke lautan spiritual mereka.

Duan Yue sangat terkejut dengan rangkaian gerakannya hingga dia tidak bisa berkata-kata. Ini bukanlah teknik seorang pejuang, tapi teknik Dewa.

Di bawah bimbingannya, ribuan kultivator mulai berkultivasi dalam diam. Seluruh lautan energi spiritual menutupi Wilayah Api, yang radiusnya ribuan mil. Segera, formasi pada bendera pertempuran api Tianluo menyebar dan tumpang tindih dengan formasi kuno Wilayah Api, saling memberi nutrisi.

Setelah melakukan semua itu, Yunxiao tersenyum dan berkata dengan tampilan yang sangat santai, “” Hehe, ini akan menarik. Bahkan jika aku berada di luar, aku dapat langsung memanfaatkan kekuatan panji perang api tianluo yang berkobar ini. Aku sangat menantikannya.”

Duan Yue menelan ludah dan memandangi lautan api yang mencapai langit. Jika dia melepaskannya, kekuatannya tidak terbayangkan. “Apa yang harus kita lakukan sekarang?” dia bertanya.
